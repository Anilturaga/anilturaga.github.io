The lifecycle of search has 2 components
1. User asks a question
2. The machine answers

There has been quite a bit of evolution of how the machine answers our questions.

1. Keyword based results
2. Results based on the semantic understanding of the question
3. Knowledge graph to bypass website links and directly provide answer
4. Generative AI that will answer the questions directly


However, little to no evolution happened on the first component where a user asks for something.

After 20+ years, we are still using the good old text input to try to express our requirement. 

More often than not, we are forced to tailer our question to the machine with assumptions about how the machine interprets our question.
